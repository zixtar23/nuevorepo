Repositorios:


¿Que es?

Un repositorio es un espacio de almacenamiento que se usa para guardar distintos tipos archivos digitales tales como videos, imagenes, documentos, animaciones, etc.Con el objetivo de gestionar, almacenar, preservar, difundir y facilitar el acceso a los objetos digitales y a la información.


¿Que tipos de repositorio hay?

Repositorio Temático: Repositorio que almacena cualquier tipo de información sin importar si esta percetene a una institución o a una persona.

Repositorio institucional: Repositorio que almacena información que es ofrecida por una institución o comunidad para facilitar la difusión de contenidos generados por ellos mismos.

Repositorio de datos: repositorios que guardan y comparten los datos de las investigaciones.


Algunos de los repositorios más comunes que se usan en el ambiente de desarrolo de software son:

  • Github
  • Mercurial
  • Bitbucket
